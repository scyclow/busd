About Burn Trade Proofs

Burnt USD ($bUSD) is a coin that can only be issued by burning a physical US Dollar bill.

Each $bUSD coin has a notional value of $1, which corresponds to the total monetary value released during an official Burn Ceremony.

$bUSD is not a stablecoin. It can never be redeemed for USD, and it may trade at values other than $1 for extended periods of time.

$bUSD is not a security. There is no utility or reasonable expectation of profit from owning it.

All $bUSD is issued on Ethereum L1.

For more information, reference the Burn Agreement.
